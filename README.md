# 🐄 SmartVet AI Ecosystem

## AI-Powered Veterinary Healthcare & Dairy Management Platform

SmartVet AI is a next-generation veterinary healthcare, disease prediction, livestock management, and dairy commerce platform designed for farmers, veterinary doctors, hospitals, and dairy consumers.

The platform combines Artificial Intelligence, Data Analytics, Veterinary Databases, Disease Intelligence, Medicine Recommendations, and Interactive Knowledge Graphs into a single ecosystem.

---

# 📌 Problem Statement

Livestock owners face several challenges:

* Limited access to veterinary doctors
* Delayed disease diagnosis
* Lack of medicine awareness
* Unavailability of withdrawal period information
* Poor animal health record management
* Difficulty finding nearby veterinary hospitals
* Lack of centralized dairy marketplace

These issues result in:

* Economic losses
* Disease outbreaks
* Reduced milk production
* Improper medicine usage
* Delayed treatment

---

SmartVet AI provides:

* Animal Registration System

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/your-repository/smartvet-ai.git
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Flask Backend

```bash
python app.py
```

Run Streamlit Frontend

```bash
streamlit run streamlit_app.py
```
